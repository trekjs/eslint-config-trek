# eslint-config-trek

A set of opinionated ESLint (http://eslint.org) rules (all rules included) tailored for Trek.js projects

## Usage

1. `npm install --save-dev eslint-config-trek`
2. create a file named `.eslintrc` in your project:

```js
{
  "extends": "trek"
  // Your overrides...
}
```
